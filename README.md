A program that takes string input and converts it into ASCII art graphical representation.
For example, if the input is "Hello" the output to the instruction {go run . "Hello\n" | cat -e} will be:
 _    _          _   _          $
| |  | |        | | | |         $
| |__| |   ___  | | | |   ___   $
|  __  |  / _ \ | | | |  / _ \  $
| |  | | |  __/ | | | | | (_) | $
|_|  |_|  \___| |_| |_|  \___/  $
                                $
                                $
